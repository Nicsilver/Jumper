# Jumper

<!-- Plugin description -->
Navigate your IDE at lightning speed. Jumper lets you jump multiple lines/items at once with custom keybindings, working seamlessly across editors, project trees, and all UI lists.

**What makes Jumper special:**
- **Universal jumping** - Works in editors, project view, Gradle view, database view, and more
- **Lightning fast navigation** - Jump 5, 10, or any number of lines with a single keystroke
- **Smart selection** - Jump with selection support in all contexts
- **Easy setup** - Modern UI with integrated keybinding configuration
- **Visual feedback** - See all your jump actions and keybindings at a glance

## Quick Start

1. Go to <kbd>Settings/Preferences</kbd> > <kbd>Tools</kbd> > <kbd>Jumper Settings</kbd>
2. Enter a jump amount (e.g., `5`) and click **Add Jump**
3. Configure keybindings for all 4 actions (Up, Down, Up+Select, Down+Select)
4. Use your new keybindings anywhere in the IDE!

Each jump creates 4 actions: Jump Up, Jump Down, Jump Up with Selection, and Jump Down with Selection.

## Installation

- Using IDE built-in plugin system:

  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>Marketplace</kbd> > <kbd>Search for "Jumper"</kbd> >
  <kbd>Install Plugin</kbd>
