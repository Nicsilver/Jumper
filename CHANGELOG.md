# Jumper Changelog

## [0.2.0] - 2026-02-07

### Major Update

- **Universal Jump Support** - Jump actions now work everywhere: editors, project view, Gradle view, database view, and all tree/list components
- **Modern Settings UI** - Completely redesigned with integrated keybinding management and live keybinding display
- **Converted to Kotlin** - Full codebase rewrite in Kotlin for cleaner, more maintainable code
- **Platform Upgrades** - Updated to IntelliJ Platform 2025.1, Kotlin 2.1.0, Java 21, and Gradle 8.13

## [0.1.5]
